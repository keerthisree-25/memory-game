# Memory Puzzle Game
A simple Memory Tile-Flipping Puzzle Game developed using Python and Tkinter.
## Description
Players flip tiles to find matching pairs. If two tiles match, they remain open.
The game ends when all pairs are matched.
## Technologies Used
- Python
- Tkinter (GUI)
## Concepts Used
- Lists
- Loops
- Conditional statements
- Event-driven programming
- GUI development
## How to Run
1. Open terminal / command prompt
2. Navigate to src folder
3. Run the command:
   python memory-game.py
